# App de Clima SPA con Vue.js

Aplicación de clima desarrollada como Single Page Application utilizando Vue.js y Vue Router.

La temática de la app está basada en distintos lugares turísticos como playas, montañas y desiertos de Chile.

## Vistas principales

Home
Muestra el listado de lugares con su clima actual.

Detalle de Lugar
Muestra información detallada del lugar seleccionado, incluyendo pronóstico y estadísticas de la semana.

## Rutas

/ → Home

/lugar/:id → Detalle de un lugar

## Interacciones

La app incluye un buscador de lugares utilizando v-model que permite filtrar el listado en tiempo real.

## Tecnologías utilizadas

Vue.js  
Vue Router  
JavaScript  
CSS

## Cómo ejecutar el proyecto

Instalar dependencias:

npm install

Ejecutar el servidor:

npm run dev
